UE5利用行为树实现简单的怪物AI

title: UE5利用行为树实现简单的怪物AI

创建行为树：
在UE5编辑器中，创建一个新的行为树资源（Behavior Tree）。可以通过右键点击Content Browser，选择”Create Advanced Asset”，然后选择”Behavior Tree”来创建一个新的行为树。
配置行为树：
打开创建的行为树资源，可以看到一个树状结构的编辑器。在这里，你可以添加各种行为节点来描述角色的行为流程。
首先，添加一个根节点（Root），这将是整个行为树的入口点。
添加一个随机选择节点（Random Selector），用于随机选择巡逻行为和追击行为。
在随机选择节点下添加两个子节点：
第一个子节点是巡逻行为，你可以使用一系列的行为节点来实现巡逻逻辑，如移动到目标位置、等待一段时间等。
第二个子节点是追击行为，你可以使用一系列的行为节点来实现追击玩家的逻辑，如移动到玩家位置、判断距离、攻击等。

实现角色的更新逻辑，用于更新黑板值：

// 在角色的更新函数中，更新黑板值
if (Blackboard)
{
    // 更新玩家位置
    ACharacter* PlayerCharacter = UGameplayStatics::GetPlayerCharacter(GetWorld(), 0);
    if (PlayerCharacter)
    {
        FVector PlayerLocation = PlayerCharacter->GetActorLocation();
        Blackboard->SetValueAsVector(TEXT("PlayerLocation"), PlayerLocation);
    }

    // 更新玩家探测状态
    bool bPlayerDetected = DetectPlayer();  // 自定义函数，用于检测玩家是否在可视范围内
    Blackboard->SetValueAsBool(TEXT("bPlayerDetected"), bPlayerDetected);
}

在AI控制器中使用行为树：
在AI控制器中，你需要加载和启动行为树，以及设置黑板值。

在AI控制器的头文件中，添加行为树的引用：

在AI控制器的源文件中，加载和启动行为树，并设置黑板值：

// 在头文件中声明以下成员变量
UBehaviorTreeComponent* BehaviorTreeComponent;
UBlackboardComponent* BlackboardComponent;

// 在构造函数或其他适当的地方初始化行为树组件和黑板组件
BehaviorTreeComponent = CreateDefaultSubobject<UBehaviorTreeComponent>(TEXT("BehaviorTreeComponent"));
BlackboardComponent = CreateDefaultSubobject<UBlackboardComponent>(TEXT("BlackboardComponent"));

// 将黑板组件分配给行为树组件
BehaviorTreeComponent->InitializeBehaviorTree(BehaviorTree);
BehaviorTreeComponent->SetBlackboard(BlackboardComponent);

// 启动行为树
BehaviorTreeComponent->StartTree(*BehaviorTree);

在行为树中使用黑板值：
在行为树中，可以使用黑板值来决策角色的行为。

在巡逻行为节点中，可以使用黑板值来获取目标位置，对应的黑板键为”PlayerLocation”。
在追击行为节点中，可以使用黑板值来获取玩家位置，对应的黑板键为”PlayerLocation”。
在追击行为节点中，可以使用黑板值来获取玩家探测状态，对应的黑板键为”bPlayerDetected”。
